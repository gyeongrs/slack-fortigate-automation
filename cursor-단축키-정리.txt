============================================================
 Cursor 단축키 정리 (Windows 기준)
 출처: Cursor 공식 Keyboard Shortcuts 도움말 / 레퍼런스
 작성일: 2026-06-04
============================================================

[참고]
- Cursor는 VS Code와 동일한 기본 단축키 + AI 전용 단축키 구조입니다.
- 공식 레퍼런스 표는 Mac 표기 위주라, 변환 규칙(Cmd→Ctrl, Opt→Alt)을
  적용해 Windows 조합으로 옮긴 항목이 있습니다.
- 실제 값은 버전/설정에 따라 다를 수 있으니, 아래 "단축키 확인/변경"
  메뉴에서 직접 확인하는 것이 가장 정확합니다.


------------------------------------------------------------
1. 핵심 AI 기능 (도움말에 Windows 명시됨)
------------------------------------------------------------
Ctrl + I  또는 Ctrl + L   사이드 패널(채팅/Agent) 토글
Ctrl + K                  인라인 편집(Inline Edit) 열기
Ctrl + .                  모드 메뉴(Mode Menu) 열기
Shift + Tab               Agent 모드 전환 (Agent/Ask/Plan 순환)
Ctrl + /                  AI 모델 순환 전환
Tab                       자동완성 제안 수락


------------------------------------------------------------
2. 채팅(Chat) 입력창
------------------------------------------------------------
Enter                     메시지 전송 (기본)
Ctrl + Enter              메시지 큐에 추가 / 입력 중 강제 전송
Ctrl + Enter              제안된 변경 사항 모두 수락
Ctrl + Backspace          제안된 변경 사항 모두 거부
Ctrl + Shift + Backspace  생성 취소
Ctrl + Shift + L          선택한 코드를 컨텍스트로 추가 (코드 선택 상태)
Ctrl + V                  클립보드 내용을 컨텍스트로 추가
Ctrl + Shift + V          클립보드 내용을 입력창에 추가
Tab                       다음 메시지로 이동
Shift + Tab               Agent 모드 순환 전환
Ctrl + Alt + /            모델 토글
Ctrl + N  또는 Ctrl + R   새 채팅
Ctrl + T                  새 채팅 탭
Ctrl + [                  이전 채팅
Ctrl + ]                  다음 채팅
Ctrl + W                  채팅 닫기
Escape                    입력창 포커스 해제


------------------------------------------------------------
3. 인라인 편집(Inline Edit)
------------------------------------------------------------
Ctrl + K                  인라인 편집 열기
Ctrl + Shift + K          입력 포커스 토글
Enter                     제출(실행)
Ctrl + Shift + Backspace  취소
Alt + Enter               빠른 질문하기(Quick question)


------------------------------------------------------------
4. 코드 선택 & 컨텍스트 추가
------------------------------------------------------------
@                         @-멘션 (파일/심볼 등 참조)
/                         단축 명령(Shortcut Commands)
Ctrl + Shift + L          선택 영역을 채팅(Chat)에 추가
Ctrl + Shift + K          선택 영역을 편집(Edit)에 추가
Ctrl + L                  선택 영역을 새 채팅에 추가
Ctrl + M                  파일 읽기 전략 토글
Ctrl + →                  제안의 다음 단어만 부분 수락
Ctrl + Enter              채팅에서 코드베이스 검색


------------------------------------------------------------
5. Tab (자동완성)
------------------------------------------------------------
Tab                       제안 전체 수락
Ctrl + →                  다음 단어만 부분 수락


------------------------------------------------------------
6. 터미널 AI
------------------------------------------------------------
Ctrl + K                  터미널 프롬프트 바 열기
Ctrl + Enter              생성된 명령 실행
Escape                    명령 수락


------------------------------------------------------------
7. 단축키 확인 / 변경 방법
------------------------------------------------------------
1) Keyboard Shortcuts 열기:
   - Ctrl + R 다음 Ctrl + S
   - 또는 Ctrl + Shift + P → "Keyboard Shortcuts" 검색
2) 변경할 명령 검색
3) 옆의 연필(편집) 아이콘 클릭
4) 원하는 키 조합 입력
5) Enter 로 저장
* 모든 AI 단축키는 여기서 자유롭게 재매핑 가능


------------------------------------------------------------
8. 버전에 따라 달라질 수 있는 부분
------------------------------------------------------------
- 일부 Windows 조합은 Mac 표기를 변환한 값이라 실제와 다를 수 있음.
- Ctrl+I / Ctrl+L 은 모드 바인딩 설정에 따라 동작이 달라질 수 있음.
- 동일 동작에 두 조합(예: 새 채팅 Ctrl+N / Ctrl+R)이 매핑된 경우가 있고
  업데이트로 기본값이 바뀔 수 있음.
- Agent/Composer 용어·레이아웃 단축키는 UI 변경에 따라 달라질 수 있음.
  (예: Agent 레이아웃 토글 Mac Cmd+E → Windows Ctrl+E)
